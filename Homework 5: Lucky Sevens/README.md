# Homework 5: Lucky Sevens

## Overview
A dice game simulation where the user bets a starting stake on rolling a pair of dice. A sum of 7 wins $4, while any other sum loses $1, and the program tracks the highest stake achieved until the user runs out of money.

## Concepts Covered
* The `random` module
* Nested `while` loops
* Input validation (checking if input is a digit)
