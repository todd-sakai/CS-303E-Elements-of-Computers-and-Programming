# Homework 3: Wind Chill Calculator

## Overview
This program calculates the wind chill based on user-provided temperature and wind speed. It takes inputs in Fahrenheit and miles per hour, applies the standard calculation formula, and outputs formatted reports in both Fahrenheit and Celsius.

## Concepts Covered
* User input and type casting (floats)
* Mathematical formulas and exponents (`**`)
* String formatting (`format(x, '9.3f')`) and unicode characters for degrees
