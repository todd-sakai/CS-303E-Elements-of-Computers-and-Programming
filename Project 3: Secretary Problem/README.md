# Project 3: Secretary Problem Simulator

## Overview
This program simulates the Secretary Problem by running experiments on generated applicant data to test the mathematical optimal stopping strategy. It reads and writes candidate ranking data from text files to evaluate the success rate of the algorithm over multiple trials.

## Concepts Covered
* File I/O operations (reading, writing, and parsing text files)
* Mathematical constants and algorithmic logic (`math.e`)
* List slicing, indexing, and maximum value tracking
* Generating and formatting statistical percentages
