# Project 2: Prime Factory

## Overview
This program provides an interactive command-line interface to perform various mathematical operations related to prime numbers. It features custom functions for determining primality, finding the Nth prime, generating lists of primes, and calculating prime factorizations.

## Concepts Covered
* Interactive menus and input validation
* Nested loops and control flow
* Algorithmic optimization (efficient prime checking using square roots)
* List manipulation and dynamic array building
