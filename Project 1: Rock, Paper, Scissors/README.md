# Project 1: Rock, Paper, Scissors

## Overview
This program allows a user to play a continuous game of Rock, Paper, Scissors against a machine opponent. The machine can make random moves or follow a predefined sequence passed via the command line, and the program calculates final win/loss game statistics.

## Concepts Covered
* Continuous game loops (`while True`)
* Conditional logic and complex boolean evaluations
* Command-line argument parsing (`sys.argv`)
* Random choice generation (`random` module)
