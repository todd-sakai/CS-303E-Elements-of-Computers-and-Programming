# Homework 7: Toy Car Simulator

## Overview
This program creates a `ToyCar` class to simulate a car moving on a 2D Cartesian grid. It tracks the car's X/Y coordinates and heading, allowing it to move forward, turn left or right, and autonomously navigate to specific coordinates.

## Concepts Covered
* Object-Oriented Programming (OOP)
* Classes, constructors (`__init__`), and methods
* State management and mutation
